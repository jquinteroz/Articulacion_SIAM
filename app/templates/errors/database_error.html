Error de {{ error_type|title }}

No se pudo conectar a la base de datos MySQL

¿Qué pasó?

  • El servidor MySQL no está ejecutándose
  • XAMPP no está iniciado o MySQL está detenido
  • El puerto de MySQL (3306) puede estar bloqueado
  • Las credenciales de conexión pueden ser incorrectas

Solución: Inicia MySQL en XAMPP

  1. Abre XAMPP Control Panel desde el menú de inicio o escritorio
  2. Busca el módulo "MySQL" en la lista de servicios
  3. Haz clic en el botón "Start" junto a MySQL
  4. Espera el indicador verde que confirma que MySQL está ejecutándose
  5. Recarga esta página o reinicia la aplicación Flask
